系统使用说明下载

下面只是简易说明

1，pom引入：
<dependency>
  <groupId>com.jd.creditfactory.email</groupId>
  <artifactId>creditfactory-email-export</artifactId>
   <version>1.0.0-20160408.030049-4</version>
</dependency>
2，Spring配置中引入jsf配置
<jsf:consumer id="p2pmcService" interface="com.jd.creditfactory.email.export.P2PMCApi" protocol="jsf" alias="p2pmcService" timeout="10000">
</jsf:consumer>
3，在需要发送邮件的类中注入 p2pmcservice,调用sendMsg方法发送邮件：
p2pmcService.sendMsg(“注册的系统编码”, “邮件标题”,“邮件内容” ,”订阅分组编码”，”告警级别” ,”应用所在服务器ip”);
